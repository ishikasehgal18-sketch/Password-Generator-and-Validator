#  Password Validator & Generator

A simple **Password Validator & Generator** built using **Python**. This project validates user passwords based on basic security rules and also includes a function to generate strong random passwords.

---

##  Features

-  Validates password length
-  Checks if the password contains at least one numeric digit
-  Uses custom exception handling for invalid passwords
- Generates strong random passwords using letters, numbers, and special characters
- Command-line interface (CLI)

##  How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Password-Validator.git
```

2. Open the project folder.

3. Run the program.

```bash
python password_validator.py
```
